# Changelog

## Wither's Wrath 1.15 (Latest)

### Compatibility
- **1.21.11+** requires changes that need a new version entirely, which is ready for release
- Waiting for full release of 1.21.11+

### Direct Changes to the Wither

#### Phase 1 Behavior (300HP-600HP)
- The Wither will now move away from players to maintain distance
- Enables more effective ranged attacks

#### Skull Improvements
- Dangerous (blue) skulls buffed with bigger explosions
- Deal significantly more damage than before
- Leave smoke clouds behind

#### Armor Adjustments
- **Phase 1**: Reduced armor (easier to damage)
- **Phase 2**: Increased armor (harder to damage)

#### Mob Conversions
- Converted Skeletons now carry bows (matching their pre-conversion state)

#### Blaze Spawning Changes
Updated spawn counts during midpoint transition:
- Fewer than 5 players: 10 Blazes
- Fewer than 10 players: 20 Blazes
- More than 10 players: 30 Blazes

### Changes to Attacks

#### 150 HP Trigger
- A swarm of smaller Wither Skeletons now spawns at 150 HP
- Helps the Wither deal with close-range attackers

#### Burst Attack
- The Wither now regularly shoots bursts of skulls directly at the nearest player
- The skull shot at players in earlier versions is now a slow fire shot marked as dangerous

### Fixes
- Fixed Wither destroying bedrock in very specific edge cases
- Reduced volume on enraged skulls (were incredibly loud at 100%)

## Wither's Wrath 1.14.2

**Release Date**: September 30, 2024

### Changes

#### Wither Detection Rework
- Completely reworked how the Wither is detected
- Now works with Withers spawned through:
  - Spawn Egg
  - `/summon` command
  - Traditional soul sand + skulls method

#### Spawn Limitations
- Only one Wither can exist at a time
- If a player tries to summon another while one exists:
  - Plays a warning sound
  - Sends message to nearest player
  - Returns the skulls and soul sand

#### Visual Improvements
- Added minimal ambient particles (ash) around the Wither

#### Compatibility
- Full compatibility with 1.21.9 (use 1.21.5+ versions)

### Fixes
- Fixed destruction toggle not working as intended (#24)
- Fixed multiplayer breaking some functions at spawn
- Improved spawn reliability in multiplayer environments

## Wither's Wrath 1.14.1

**Release Date**: September 6, 2024

### Changes

#### Midpoint Mechanics
- Midpoint now throws players up based on proximity to the Wither
- Closer players get launched higher
- Wither only triggers the slam when it's on the ground (removed time-based trigger)

#### Mob Equipment
- Skeletons with bows now have a named bow from Mickey Joe's "Barely Default" resource pack

### Fixes
- Fixed gamerules being overridden during Wither fights (#23)
- Fixed some selectors having wrong format
- Fixed trail particle at the start of the fight

### Important Note
- **Mod versions** no longer have the resource pack pre-installed
- Download resource pack separately from [Modrinth](https://modrinth.com)

## Wither's Wrath 1.14

**Release Date**: June 25, 2024

### Changes

#### Enraged Mode Skull Rework
Complete rework of enraged mode skull behavior:
- Skulls now shoot upward initially
- Pause mid-air
- 1.5 seconds later, shoot toward the nearest player
- Each skull tracks independently

**Video demonstration**: `enragedskulls.mp4`

### Added

#### Direct Skull Attack
- 20 seconds after the Wither is spawned
- Shoots a high-speed direct Wither skull at a random player

### Fixes
- Fixed items getting killed by thunder when the Wither is killed
- Fixed multiplayer summon breaking (bug present since release 4 years ago)
- Fixed some names in wrong format
- Fixed Wither dropping armor on death due to incorrect command formatting in 1.21.5+
